### 🚀 Experiência Recente com Cloud, Segurança e Performance
Em meus últimos projetos, implementei:
* **Desenvolvimento Moderno:** Criação de interfaces dinâmicas e performáticas com **React** e **Next.js**.
* **Autenticação Segura:** Uso de `jsonwebtoken` e `bcrypt` para proteção de dados e senhas.
* **APIs de Alta Performance:** Consumo de dados com `axios` em ambientes `express`.
* **Banco de Dados na Nuvem:** Modelagem e integração com `mongoose` (MongoDB Atlas).
* **Deploy Automatizado:** Hospedagem de aplicações na **Vercel** e **Render**.
